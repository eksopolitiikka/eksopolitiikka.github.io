Project UFO on NBC:n l\u00e4hett\u00e4m\u00e4 TV-sarja joka kesti kaksi kautta vuosina 1978 ja 1979. Sarja perustui USA:n ilmavoimien Project Blue Bookiin<\/a>. Sarjan luoja oli Dragnet-veteraani Jack Webb, joka k\u00e4vi l\u00e4pi ilmavoimien UFO-tiedostoja etsien ideoita jaksoille. Jokaisessa jaksossa oli dramatisoitu yksi UFO-tapaus. Project UFO ennakoi monia Salaisten Kansioiden teemoja, ilman sen romanttista vivahdetta tai hallituksen vastaista (tai my\u00f6sk\u00e4\u00e4n avaruusolioiden vastaista) paranoiaa. Monet Blue Bookin (ja Project UFOn) havainnoista pystyttiin selitt\u00e4m\u00e4\u00e4n luonnollisesti. Jotkut kuitenkin j\u00e4tettiin selitt\u00e4m\u00e4tt\u00e4, ja t\u00e4m\u00e4 viittaisi avaruusoliokontaktiin.<\/p>\n
