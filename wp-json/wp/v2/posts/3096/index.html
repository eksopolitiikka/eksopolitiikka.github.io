Roman Tregubov todisti massiivisen valkoisen massiivisen valopallon liikkeit\u00e4 Novosibirskiss\u00e4 ennen sen katoamista metsikk\u00f6\u00f6n.<\/p>\n

<\/a><\/p>\n

Roman onnistui kuvaamaan harvinaisen ilmi\u00f6n.<\/p>\n

Videolla k\u00e4yd\u00e4\u00e4n seuraava keskustelu:<\/p>\n

Nainen: \u201cMik\u00e4 tuo on tuolla?\u201d
\nRoman: \u201cTulipallosalama\u201d
\nMies: \u201cKatso, katso, mik\u00e4 tuo on?\u201d
\nRoman: \u201cArgh, se menee pois. N\u00e4en sellaisen ensimm\u00e4isen kerran el\u00e4m\u00e4ss\u00e4ni. Se karkaa. Kadotin sen. Miss\u00e4 se on? Tuolla se on. Katoaakohan se kohta. Jep, se katosi.”<\/p>\n

Roman Tregubov selitti Siberiantimes:lle <\/a><\/u>ett\u00e4 h\u00e4nen t\u00e4ytyi seist\u00e4 tiilten p\u00e4\u00e4ll\u00e4 saadakseen paremmin kuva yli aidan ja pusikoiden,mink\u00e4 takia kuva heiluu.<\/p>\n
