Silminn\u00e4kij\u00e4 kertoo: Aurinkoisena p\u00e4iv\u00e4n\u00e4 Middlesexin piirikunnassa New Jerseyss\u00e4 elokuussa 2018 se n\u00e4ytti kuin kotelolta joka ajelehti ilmassa. Se oli materiaa, ei valoa. Se oli valkoinen auringonvalossa.<\/p>\n

<\/a><\/p>\n

Pinta n\u00e4ytti sile\u00e4lt\u00e4, se ajelehti minua kohti ja sitten pois. Luulin ett\u00e4 olin tulossa hulluksi, sitten se palasi kohti minua, pys\u00e4htyi ja meni pois.<\/p>\n

Ei ole ideaa siit\u00e4 mik\u00e4 se voi olla. Ehk\u00e4 hy\u00f6nteinen, jonkinlainen untuva tai h\u00f6yhen kukista tai puista. Tai ehk\u00e4 henkinen orbi tai sitten \u00e4lyk\u00e4s el\u00e4m\u00e4nmuoto?<\/p>\n
